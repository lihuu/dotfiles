# AI CLI Tools

This directory contains installation and configuration for various AI terminal agents.

## Tools Included

- **Claude Code**: `@anthropic-ai/claude-code` - The official Anthropic CLI agent.
- **Qwen Code**: `@qwen-code/qwen-code` - An open-source agent optimized for Qwen3-Coder.
- **Gemini CLI**: `@google/gemini-cli` - The official Google Gemini CLI agent.

## Installation

To install all tools globally using `npm`:

```bash
./install.sh
```

The script uses `npm install -g --force` to ensure existing binaries are updated correctly.
